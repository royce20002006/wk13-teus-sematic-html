My HTML Cheat Sheet

Headings

Headings from `h1` to `h6` identify blocks and sections. Every page must have one `h1`.

Paragraphs

Paragraphs of text go in `p` tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in `ul` tags. Each list item goes in a `li` tag.

Types of lists:

  • Unordered lists (`ul`)
  • Ordered lists (`ol`)

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each list item goes in a `li` tag.

Polya's Problem Solving Framework

  1. Understand the problem
  2. Come up with a plan
  3. Carry out the plan
  4. Go back and improve your solution

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the `href` attribute. Go to DuckDuckGo to search for more information.

Setting `target="_blank"` will open the link in a new window. See MDN for more details.

Images

You can use the img tag to add images to your page.

Tables

You can use tables to showcase data that is important using `table` then you will need a `thead` and `tbody`. you can also use a `tfoot` also `tr` stands for row. then `th` which signify whether it is a column or row header make sure and put that in the scope attribute. put `td` to hold the data.

Forms

Forms are very important things to remember because if you want to have a user registration or a simple survey or contact form you need to be able to do that the syntax for that is first you have to have `form` followed by a `fieldset` and a `legend` then use `p` and inside of them you could have anything from `input with different types and `label` and also `select` or even `textarea`.
